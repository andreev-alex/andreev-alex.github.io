Federal Bank's Rewards Program for 73M+ users

'Sber Spasibo' is a rewards program in Eastern Europe that caters to an active user base of 73 million individuals. It helps them save money on their everyday purchases.

About project

Problem & Personal Contribution & Impact

The problem behind this project was related to an issue with using the rewards program inside Sber devices. Here's how it looks: when our users want to use the rewards program on Sber Devices, they need to go outside the application through WebView, which takes them to the website.

As a Senior Product Designer on this project, my primary focus was on developing a cross-platform design that seamlessly integrated across various platforms, including iOS, tablet, and Smart TV. This would allow Sberbank customers to fully interact with the "Sber Spasibo" rewards program without leaving the "Sber Salut" app.

As a result of this work, users are able to use rewards program via Sber devices which leads to 10% increase in engagement rate.
